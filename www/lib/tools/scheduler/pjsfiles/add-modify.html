신규 작업 추가
실행파일: 경로 확인[^]
실행할 파일의 위치를 적어주세요. - help and more detail here[^]
현재 위치 :
작업 이름:
작업 이름을 적어주세요.
시작 일시:
작업을 새로 시작할 날짜와 시간을 적어 주세요. 예) 2012-02-01 23:01:01
시작 일시에 적힌 시간부터 작업이 시작됩니다.
입력하지 않으면 마지막 실행 시간이나 저장되는 시간부터 작업 실행 간격에 맞게 다음 실행 일시를 계산하여 추가 됩니다.


작업 실행 간격:
분 별
 - 매 분마다 실행
시간별
 - 매 시간마다 실행
일별
 - 매 번째 일마다 실행
주별
 - 매 번째 주마다 실행
30일마다 실행해야 한다면 4주와 2일을 선택하세요. 4 * 7 + 2 = 30
종료 방법:
한번만 실행을 선택하면 한번 실행 후 작업이 자동 삭제됩니다.
